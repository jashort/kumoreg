# KumoReg

Convention Registration Management

- At-Convention and pre-registered attendee checkin
- Badge printing
- Reporting
- Roles and permission management
    - Allow only certain users to search for / edit / check in attendees / set prices manually


# Requirements
- Java 8
- MySQL
- Tomcat
